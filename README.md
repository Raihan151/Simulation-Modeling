<h1> NumPy Mathematical Operations and Matrix Analysis</h1>

<p>
This project demonstrates the use of <b>NumPy</b> to perform a wide range of
<b>mathematical, trigonometric, statistical, and matrix operations</b>
on scalars, vectors, and matrices.
The code was written and executed using <b>Google Colab</b>.
</p>

<hr>

<h2> Libraries Used</h2>
<ul>
  <li><b>NumPy</b> – Numerical computing and array operations</li>
  <li><b>SymPy</b> – Symbolic mathematics (imported for extensibility)</li>
  <li><b>Matplotlib</b> – Data visualization (optional)</li>
  <li><b>Seaborn</b> – Statistical visualization (optional)</li>
</ul>

<pre>
pip install numpy sympy matplotlib seaborn
</pre>

<hr>

<h2> Features Implemented</h2>

<h3> Operations on 1D Array (Vector)</h3>
<ul>
  <li>Trigonometric functions: sin, cos, tan</li>
  <li>Inverse trigonometric functions: arcsin, arccos, arctan (with normalization)</li>
  <li>Exponential and logarithmic functions</li>
  <li>Absolute value and square root</li>
  <li>Remainder calculation</li>
  <li>Rounding functions: round, floor, ceil</li>
</ul>

<h3> Operations on 2×2 Matrix</h3>
<ul>
  <li>Element-wise trigonometric operations</li>
  <li>Inverse trigonometric functions with clipping</li>
  <li>Exponential and logarithmic calculations</li>
  <li>Remainder and rounding functions</li>
</ul>

<h3> Operations on 3×2 Matrix</h3>
<ul>
  <li>Trigonometric and inverse trigonometric operations</li>
  <li>Exponential and logarithmic functions</li>
  <li>Absolute value and square root</li>
  <li>Rounding and remainder calculations</li>
</ul>

<h3> Statistical Analysis on 2×3 Matrix</h3>
<ul>
  <li>Maximum and minimum values with indices</li>
  <li>Vector length (total elements)</li>
  <li>Row-wise descending sort</li>
  <li>Entire matrix descending sort</li>
  <li>Sum and product of elements</li>
  <li>Mean, median, and standard deviation</li>
</ul>
